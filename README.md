# Snake Game en la Terminal

Este es un simple juego de Snake que se ejecuta en la terminal utilizando caracteres ASCII para la representación gráfica. El objetivo del juego es controlar una serpiente para que coma comida (representada por un carácter) y evitar chocar contra las paredes o contra sí misma.

## Características

- **Interfaz basada en caracteres ASCII:** El juego se desarrolla completamente en la terminal y utiliza caracteres ASCII para representar la serpiente, la comida y el tablero.
  
- **Control simple:** El jugador puede controlar la dirección de la serpiente utilizando las teclas de dirección (arriba, abajo, izquierda, derecha) o las teclas W, A, S y D.

## Requisitos del sistema

- Python 3.x
- Sistema operativo compatible con la ejecución de aplicaciones de terminal

## Cómo jugar

1. Clona este repositorio en tu máquina local.

    ```bash
   git clone https://github.com/caiv2303/snake.git

2. Navega hasta el directorio del juego.

    ```bash
    cd snake

3. Ejecuta el juego.

   ```bash
    python snake.py

4. Utiliza las teclas de dirección
(arriba, abajo, izquierda, derecha) o las teclas W, A, S y D para controlar la dirección de la serpiente.

5. Trata de comer la comida (representada por el carácter @) sin chocar contra las paredes o contra la serpiente misma.

6. El juego termina cuando la serpiente choca contra una pared o contra sí misma.
